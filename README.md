# The Gooners

Ein minimalistisches 3D-First-Person-Browserspiel mit Three.js. Du schießt weiße Spritzen auf Bots, die dich verfolgen.

## Features
- First-Person-Steuerung (WASD, Maus, Sprint, Springen, Pause)
- Schießen mit Projektil-Pooling (weiße Spritzen)
- Bots mit einfachem KI-Verhalten (Patrouille, Verfolgung, Angriff)
- HUD mit HP, Score, Bots
- Spielzustände: Start, Spiel, Pause, Game Over
- Performant und gut strukturiert, leicht erweiterbar

## Setup
1. Projektordner in VS Code öffnen
2. `index.html` mit Live Server starten
3. Klicken zum Starten und Maus sperren
